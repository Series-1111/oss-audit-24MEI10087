## Project Description
This project is an audit of the open-source software Git. It includes analysis of its origin, license, ecosystem, and comparison with proprietary tools. It also includes five shell scripts demonstrating Linux and open-source concepts.

## Scripts

### Script 1: System Identity Report
Displays system information like OS, kernel, user, uptime, and license.

### Script 2: FOSS Package Inspector
Checks if Git is installed and displays its details.

### Script 3: Disk and Permission Auditor
Analyzes directories and displays permissions and sizes.

### Script 4: Log File Analyzer
Counts occurrences of keywords in system log files.

### Script 5: Open Source Manifesto Generator
Generates a personalized open-source statement.

## How to Run

1. Open terminal in Linux
2. Make scripts executable:
   chmod +x script1.sh script2.sh script3.sh script4.sh script5.sh

3. Run scripts:
   ./script1.sh
   ./script2.sh
   ./script3.sh
   ./script4.sh /var/log/syslog
   ./script5.sh

## Requirements
- Linux (Ubuntu recommended)
- Git installed
